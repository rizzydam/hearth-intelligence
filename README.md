# Hearth Intelligence Patterns

Reusable intelligence pattern designs extracted from the [Home](https://github.com/rizzydam/home) ecosystem. These patterns are context-independent — they work in any system that monitors, analyzes, or coaches humans through data.

## Patterns

| Pattern | What It Solves |
|---------|---------------|
| [Actionable Intelligence](patterns/actionable-intelligence.md) | Turning analysis into action — what happened, why it matters, what to do |
| [Behavioral Baselines](patterns/behavioral-baselines.md) | Establishing "normal" for an individual, detecting meaningful deviations |
| [Crisis Escalation](patterns/crisis-escalation.md) | Knowing when AI should step back and suggest professional help |
| [Compound Stress](patterns/compound-stress.md) | Detecting stress cascades across multiple life domains |
| [User Coaching](patterns/user-coaching.md) | Teaching users to interpret their own data, not just showing results |
| [Adaptive Communication](patterns/adaptive-communication.md) | Adjusting tone, complexity, and emotional register by context |
| [Group Dynamics](patterns/group-dynamics.md) | Understanding group health, participation equity, shared decision-making |
